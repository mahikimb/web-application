# Public Images Directory

## Team Profile Pictures
Place team member profile pictures in the `team/` folder.

**File naming convention:**
- `sarah-johnson.jpg` (or .png)
- `michael-chen.jpg`
- `emily-rodriguez.jpg`
- `david-kim.jpg`
- `joseph-mwaka.jpg`
- `lucas-mahikimba.jpg`

**Recommended:**
- File format: JPG or PNG
- Recommended size: 400x400px (square images work best)
- All images should be the same size for consistency

**Note:** Images in the `public` folder can be accessed directly via URL paths like `/images/team/sarah-johnson.jpg`
