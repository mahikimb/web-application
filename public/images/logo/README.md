# Logo Image

Place your Softica Labs logo image in this folder.

**File naming:**
- `logo.png` (preferred)
- `logo.svg` (alternative)
- `logo.jpg` (alternative)

**Recommended:**
- File format: PNG or SVG
- Recommended size: At least 200x200px for best quality
- Transparent background works best

The component will automatically try to load the logo in this order:
1. logo.png
2. logo.svg
3. logo.jpg

If none are found, it will display the default SVG logo.
