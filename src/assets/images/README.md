# Images Directory

## Logo Image
Place your Softica Labs logo image in the `logo/` folder.

**Recommended:**
- File name: `logo.png` or `logo.svg`
- Supported formats: PNG, SVG, JPG, JPEG
- Recommended size: At least 200x200px for best quality

## Team Profile Pictures
Place team member profile pictures in the `team/` folder.

**File naming convention:**
- `sarah-johnson.jpg` (or .png)
- `michael-chen.jpg`
- `emily-rodriguez.jpg`
- `david-kim.jpg`
- `lisa-anderson.jpg`
- `james-wilson.jpg`

**Recommended:**
- File format: JPG or PNG
- Recommended size: 400x400px (square images work best)
- All images should be the same size for consistency

## How to Add Images

1. Copy your logo image to: `src/assets/images/logo/logo.png` (or .svg)
2. Copy team member photos to: `src/assets/images/team/` with the names listed above
3. The components will automatically use these images once they're in place
